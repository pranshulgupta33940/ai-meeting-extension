# MeetSummarizer AI

An AI-powered Google Meet summarization extension.

## Setup Instructions
1. Download this repository.
2. Ensure you have `jsPDF` downloaded in `chrome-extension/lib/jspdf.umd.min.js`. (Note: This was automatically fetched if you ran the setup process).
3. Open Chrome and go to `chrome://extensions`.
4. Enable **Developer mode** in the top right.
5. Click **Load unpacked** and select the `chrome-extension` folder.
6. Click the extension icon and hit the **gear icon** or right-click and select **Options**.
7. Enter your **Claude API Key** and **EmailJS Keys**.

## Usage
1. Open a Google Meet room.
2. **CRITICAL:** You must click the **Turn on captions (CC)** button in Google Meet for the extension to capture the transcript.
3. Open the extension popup. You should see "Capturing" with a green pulse and live text updates.
4. Add emails, hit "Summarize & Email PDF".
5. Alternatively, after summarizing, click "Download PDF".

## EmailJS Setup
1. Create a free account at [EmailJS.com](https://emailjs.com).
2. Add a new service (e.g., Gmail). Note the **Service ID**.
3. Create a new Email Template. Note the **Template ID**.
4. Set the "To Email" field to `{{to_email}}`.
5. In your Email Template Body, paste the following:

```html
<div style="font-family: sans-serif; max-width: 600px; margin: 0 auto; color: #1e293b;">
  <h2>Meeting Summary: {{meeting_title}}</h2>
  <p style="color: #64748b;">Date: {{meeting_date}} | Key Points: {{key_points_count}}</p>
  <hr style="border: none; border-bottom: 1px solid #e2e8f0; margin: 20px 0;" />
  
  {{{meeting_summary}}}
  
  <br />
  <p style="font-size: 12px; color: #94a3b8;">Generated by MeetSummarizer AI</p>
</div>
```
*(Note the triple brackets `{{{meeting_summary}}}` which tells EmailJS to render HTML instead of plain text).*

6. Get your **Public Key** from EmailJS Account Settings.

## CSS Selectors (For Developers)
If Google Meet changes their DOM structure, captions may stop capturing. You will need to inspect the Google Meet page while captions are running and update `content.js`:

**Current Speaker Name Selectors:**
- `.zs7s8d`
- `[jsname="bVHCE"]`
- `.KcIKk`
- `.YTbUzc`

**Current Caption Text Selectors:**
- `.a4cQT`
- `[jsname="tgaKEf"]`
- `.CNusmb`
- `.jbhGbc`

To find new ones:
1. Turn on CC in Google Meet.
2. Right-click the caption and Inspect.
3. Find the container holding the speaker name and note its class or jsname.
4. Find the span holding the actual text and note its class or jsname.
5. Add to `content.js`.
